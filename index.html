2. Dress Code

All employees are expected to dress business casual. On Fridays, jeans are permitted. Please ensure that all safety gear is worn when entering the manufacturing floor.

IT: Server Maintenance

Servers must be rebooted monthly. Ensure that the backup javascript functions are running correctly before initiating a restart sequence.

Check the RAID controller status and ensure all drives are green.

IT: Security Protocols

Passwords must be changed every 90 days. Two-factor authentication is required for all remote access VPN connections.
